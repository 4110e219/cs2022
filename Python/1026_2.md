# Exercise Fibbonaci Sequence
- Iterative
```python
def fib(n):
   if n ==1:
     return 1
   else:
     return fib(n-1) + n

fib(7)
# 1+2+3+4+5+6+7 = 28
```
- Recursive
```python
def fib(n):
   result = 0; 
   for i in range(n+1):
        result += i
   return result     

fib(7)
```
