## Example 2: remove() method on a list having duplicate elements
```python
# names list
names = ['joana', 'grace', 'rogie', 'cleo', 'jamaica']

# 'rogie' is removed
names.remove('rogie')


# Updated names list
print('Updated names list: ', names)
```
